好处是，国外的不需要实名，也不需要申请短信内容模板，免费次数多，python调用十分饭方便。

坏处是因为这免费的500条短信测试使用，只能同时给一个手机号发，并且发送之前需要短信验证。

网址：https://www.twilio.com/
