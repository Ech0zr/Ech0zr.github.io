安装apache2

安装mysql

mysql安全配置

是否安装验证密码插件

怕麻烦选择N，另外只要密码安全就没啥。

是否禁用远程root登陆 Y

重新加载权限表 Y

安装php

修改首先访问php文件

打开/etc/apache2/mods-enabled/dir.conf文件并将其更改为首先列出index.php。

重启Apache

例如安装curl

在查找所需的模块时遇到问题，只需输入apt install php-（模块的第一个字母），然后点击TAB键。Ubuntu将显示所有匹配的包。
